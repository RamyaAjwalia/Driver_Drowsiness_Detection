Introduction & Motivation

Problem Statement

  • Road safety is a major global concern
  • Driver drowsiness and distraction contribute significantly to road accidents
  • Need for proactive, non-intrusive monitoring systems

Solution Approach

  • Real-time computer vision system
  • Analyzes facial landmarks in live video feeds
  • Detects signs of decreased alertness:
    • Prolonged eye closure (drowsiness)
    • Head pose deviations (distraction)

Impact

  • ✓ High societal value: Contributes to safer roadways
  • ✓ Non-intrusive: No physical sensors required
  • ✓ Real-time: Suitable for Advanced Driver Assistance Systems (ADAS)

Methodology - Technical Approach

1. Facial Landmark Detection

  • MediaPipe Face Mesh: Detects 468 facial landmarks in real-time
  • Extracts key regions: eyes, mouth, facial structure

2. Feature Extraction

Eye Aspect Ratio (EAR)

  • Measures eye opening/closing
  • Low EAR indicates closed eyes → Drowsiness
EAR = (|p2-p6| + |p3-p5|) / (2 × |p1-p4|)

Head Pose Estimation

  • Calculates pitch, yaw, and roll angles using cv2.solvePnP
  • Detects head deviations from forward-facing position
  • Angles > 45° for > 5 seconds → Distraction

3. Detection Logic

  • Rule-based classification using configurable thresholds
  • Consecutive frame counting to avoid false positives
  • Blink filtering (ignores normal blinks < 1 second)

System Implementation

Real-time Processing

  • Live camera feed processing at 30 FPS
  • Efficient MediaPipe pipeline
  • Low latency detection

Multi-indicator Detection

  • ✓ Drowsiness: Eye closure > 1 second
  • ✓ Distraction: Head turn > 5 seconds
  • ✓ Smart filtering: Ignores normal mirror checks

Alert System

  • Visual Alerts: Colored borders, on-screen warnings
  • Audio Alerts: Continuous beeping with escalation
    • After 5 seconds: 3x faster and higher frequency
  • Statistics Tracking: Frame counts, alert history

Flexible Configuration

  • Adjustable thresholds (EAR, head pose angles)
  • Configurable frame counts
  • Command-line arguments for customization
